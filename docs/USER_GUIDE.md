# User Guide: Natural Language Search

## Welcome to the Marketing Content Portal!

This guide will help you search through 636+ pieces of marketing content using natural language queries.

---

## How It Works

Instead of remembering exact titles or browsing through hundreds of items, just ask questions in plain English!

**Example**:
- Instead of: *Filtering by Type="Customer Story" AND State="NV"*
- Just type: *"Show me customer stories from Nevada"*

The system uses AI (Claude or GPT) to understand your question and find the right content.

---

## Quick Start

### 1. Basic Search

Type any keyword in the search box:

```
SchoolLinks
```

This will find all content mentioning "SchoolLinks" in the title, summary, platform, or tags.

### 2. Natural Language Queries

Ask questions naturally:

```
What customer stories do we have from Nevada?
```

```
Find all videos about student engagement
```

```
Show me ebooks and webinars
```

### 3. Use Filters

Check the boxes below the search to filter by:
- Content Type (Customer Story, Video, Blog, etc.)
- State (NV, NH, SC, etc.)

### 4. View Results

Each result shows:
- **Type badge**: What kind of content it is
- **Title**: Name of the content
- **Summary**: Brief description (first 200 characters)
- **Links**: 
  - "View Live" - See the published version
  - "Download" - Get the PDF/resource directly

### 5. Export Data

Click "Export CSV" to download all results as a spreadsheet.

## Understanding Results

### Content Types

- **Customer Story**: Case studies and success stories from schools/districts
- **Video**: Full-length videos (product demos, testimonials, etc.)
- **Video Clip**: Short video segments
- **Blog**: Blog posts and articles
- **Ebook**: Downloadable guides and comprehensive resources
- **Webinar**: Recorded webinars and presentations
- **1-Pager**: Quick reference sheets and one-page summaries
- **Press Release**: Official announcements and news
- **Award**: Industry recognition and awards
- **Landing Page**: Dedicated marketing pages
- **Asset**: General marketing materials

### Link Types

- **Live Link**: Published, public-facing content
- **Ungated Link**: Direct download (often PDFs on HubSpot)

## Troubleshooting

### "No results found"

**Try**:
1. Remove filters and search again
2. Use different keywords
3. Check spelling
4. Try broader terms ("schools" instead of "school districts")

### Results seem wrong

**Try**:
1. Use more specific keywords
2. Add content type filters
3. Rephrase your query
4. Use quotes for exact phrases

### Search is slow

- AI natural language processing takes 1-2 seconds
- This is normal for complex queries
- Simple keyword search is faster (no AI needed)

## Frequently Asked Questions

**Q: Do I need to use exact titles?**  
A: No! The AI understands intent. "Nevada schools" will find "How Nevada School Districts..."

**Q: Can I search multiple states at once?**  
A: Yes! Type: "Show me content from Nevada, New Hampshire, and South Carolina"

**Q: What if I don't know the exact topic name?**  
A: Use related keywords. The AI understands context.

**Q: Can I save my searches?**  
A: Not yet, but you can export results to CSV and save locally.

**Q: How often is the data updated?**  
A: Check with your admin. Can be real-time, daily, or manual.

**Q: Can I edit content from here?**  
A: No, this is read-only. To edit, go to the source (Google Sheets, CMS, etc.)
